# Introduction

BNS (Blockchain Node Service) offers comprehensive full-node RPC call capabilities to web3 customers, leveraging blockchain nodes. This service enables clients to achieve full node hosting, eliminating the need for local node requests. Instead, customers can simply send requests to BNS, which provides an identical API, granting access to blockchain information.

By utilizing BNS, customers benefit from:

1. Exemption from node operation and maintenance costs
1. Elimination of time-consuming development processes
1. Enhanced service availability and scalability

This solution streamlines blockchain interaction, allowing customers to focus on their core business while relying on BNS for efficient and reliable blockchain node services.
